# VLAN-with-trunk

VLAN: a logical way to divide one physical network into multiple separate networks.
Why VLAN 
Without vlan the network becomes more crowded , and broadcast causes noise to others, so with vlan we can separate a network to groups.
Trunk is a port you must turn it inside the vlan if you use more than one switcher.
VLAN 10   192.168.10.0 /24
VLAN 20   192.168.20.0 /24
VLAN 30   192.168.30.0 /24
NOTICE: This must open in cisco packet tracer
